# Portfolio

This is a portfolio based on Code + Command Line theme. Developed using Figma, Next.JS, TypeScript, Tailwind, Animate.css and i18next

## How to run

```javascript
// install dependencies
npm i

// run project
npm run dev
```

or

```javascript
// install dependencies
yarn

// run project
yarn dev
```

```
Access localhost:3000
```
